爬取申万指数的初衷

做交易、分析师或者其他做宏观研究的人对指数数据都有需求。
公司有同花顺和万德终端，倒是可以很方便的获取到行业数据，但存在一个问题，就是更新速度！前者甚至有时候到晚上才能更新到当日最新数据。还让不让人下班了？！！

于是乎，我在网上找到申万的指数数据来源，每天收盘之后把数据抓下来就好喽~不过申万指数官网除了一级行业指数之外，其他的好像都要有权限才行。
