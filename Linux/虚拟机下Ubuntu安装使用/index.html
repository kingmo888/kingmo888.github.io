更新加速以及更改支持中文

更新加速

当前已经安装完毕英文原版Ubuntu，现在我们开始安装中文支持。

因为其下载都是默认在欧洲的地址，对我们国内来说速度是比较慢的，我们可以选用国内的加速点，因为阿里云的速度很好（本身阿里云的实例访问其镜像资源时也不会占用公网流量）

1、右上角的设置符号>>System Settings>>SoftWare&Updates

依然会弹出密码验证，输入你的登录密码。

之后会提示是否在当前镜像网站上查找可用更新，如图：

果断reload。

至此，更新加速已经设置完毕。

支持中文

依然是右上角的设置符号>>System Settings>>Language Support

选择[install/remove languages]（安装或者移除语言包），

然后一路回车，直至安装完毕（直接按到/dev/sr0 …），然后重启。
参考百度经验如何给Ubuntu 安装Vmware Tools

中文输入法

ubuntu安装中文输入法其实很简单的（题主其实断断续续弄了3个多小时，其实真的很简单！！！重启即可！）
下载搜狗输入法的linux版，
，双击安装。
系统设置>>语言支持>>键盘输入方式系统>>选择fcitx>>重启。

再次进入系统就看到右上角的小企鹅了。点小企鹅，配置，就可以像windows一样配置默认输入法。

此时，出现了汉语选项，拖动汉语条目到第一条，重启。

启动后，会提示是否将旧的标准文件名字改名为中文，这里不要修改，否则会出问题（可以自己试试哈）

大功搞成！

后续设置

安装Vmware Tools 自动调整虚拟机分辨率等

可是现在分辨率还是超级小，咋办呢？需要安装vmware Tools,在虚拟机列表中的虚拟机实例上右键选择安装VMWARE Tools（会加载到光驱）。
如果提示【正在进行简易安装时,无法手动启动VMware Tools安装】，把虚拟机的属性中的光驱、软驱设置为自动检测。

加载后，赚到光驱位置，右击VMwaretools-9.6.0-1294478.tar.gz，提取到，选择一个文件位置，然后通过命令行终端，赚到解压的位置，

重启，搞定。
